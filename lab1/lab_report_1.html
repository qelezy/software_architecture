Цель работы

Познакомиться с базовыми структурами данных и алгоритмами в языке Python.

Задание

Написать программу на Python, которая считает сумму чисел в списке и выводит результат на экран.

Теоретические сведения

Списки в Python — это изменяемые последовательности элементов, позволяющие хранить данные разных типов. Основные операции: добавление, удаление, обход элементов с помощью циклов.

Описание средств разработки

Использовалась среда разработки VS Code с интерпретатором Python 3.10. Программа запускалась на персональном компьютере под управлением Windows 10.

Текст программы и результаты выполнения

numbers = [1, 2, 3, 4, 5]
total = sum(numbers)
print("Сумма чисел:", total)

Анализ результатов выполнения программы

Программа корректно вычислила сумму элементов списка. Использование встроенной функции sum() упрощает обработку данных и минимизирует вероятность ошибок.

Выводы

Задача успешно выполнена: изучены списки и базовые функции Python.
